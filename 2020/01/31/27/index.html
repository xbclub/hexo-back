在命令行里的输出

  1. 调用sys模块(sys包)

    import sys
    sys.stdout.write(‘Hello!This is IDLE!’)

  1. print 语句

    print(‘Hello1 This is IDLE!’)

结果分别为 可见

  1. 引入或导入sys模块，书写输出语句，会返回语句内容，并统计字符数！括号内的内容须为字符型！包括空格和符号！！！。
  2. 直接使用print语句，则直接输出所填的内容。
  • 值得注意的是
  1. stdout只能输出字符串，如果要输出数字，必须先转成字符串的形式才可以；而print可以直接输出各种类型。

  2. print语句执行的是一个写操作，把输入的数据写到了stdout流，并进行了一些特定的格式化。所以，print实际上就是调用了sys.stdout.write()，两者等价。

外面的编译器

对于自带的python编译器，可能在用户的交互性，方便性等其他因素表现的并不是很好，因此，我们需要的其他的编译器来帮助我们！

  • 如果你在本校内，处于校园网条件下，可访问42Team下载站http://172.24.4.199 ，搜索下载这几个编译器

  1. Jetbrains PyCharm

  2. Microsoft Visual Studio

  3. Microsoft Visual Studio Code

  • 对于PyCharm和Visual Studio来说有付费专业版，而Visual Studio Code则是提供免费的使用。而前两者是在已安装好python开发环境下，软件自动完成(部分)编译器内的配置，也可手动配置；而最后者需要读者手动配置，里面提供了许多的插件库，可以自定义安装配置

笔者都使用过这三个软件，下面来进行简单的优缺点说明分析

  • 对于JetBains PyCharm

优点

  1. 从安装完成后，软件配置基本是软件本身自动配置。你只需要关注你的项目名和存放位置。很方便，还剩时！！！
  2. 提供流行的框架，看自己的项目框架选择自动或手动配置。
  3. 有丰富的插件库。可以在里面安装流行的插件。比如，在pycharm里写Java，也是可以的。只要去官方的插件网站上下载对应编译器的插件安装，配置就可以写了。
  4. 代码自动补全和自动提示。这个真的是爽的不行啊！！！这也是jetbrians全系软件的特色啊。每当自己写代码忘记或遗漏时，它会提醒你，真的是太方便了！
  5. 诸多好处，你用了就知道

缺点

  1. 专业版不免费，需要支付一定的费用。之前说了那么多的好处，提供很多便利，也可以说是人家靠这些吃饭的，付钱是肯定的！

去官网看了下，199美刀，按目前1美元~=7.x元人民币的汇率来看，每年大约需要支付1400元左右的费用！

  1. 软件安装和启动时，要占用大量CPU资源和(运行)内存空间！没其他意思，处理器不行或运行空间不够的话，要么机器太老，建议不要安装！否则运行时，就是考验你忍耐度的时候了！卡顿也是正常！
  2. 使用时，你在找找吧。我觉得就以上两个明显一点
  • 对于Microsoft Visual Studio

优点

  1. 大部分版本免费，专业版也须付费(同上一个)。它是基本是各大编译器(IDE)票选网站上第一的位置，用户使用还不错。
  2. 提供大部分的开发环境，开袋即食。只要你选好了要开发的语言或项目，里面它会给你配置好！
  3. 微软Microsoft自家产品，对Windows本身很友好。毕竟是一家人，该关照的地方不会用太大问题。我可没说JetBrains不好啊！

缺点

  1. 软件安装时体积庞大，占存储空间。官网新版本是一个在线安装器，你等它下载完，让你选安装选项时，你选了太多的选项，体积的真的吓人。一次下载使用时，我完成基本自己使用配置后，体积竟达6个G，太吓人了
  2. 可供选择很多的编程项目和语言，不太适合做微软家以外的东西(笔者自己观点)。

官网截图 绝大部分是Windows自己的相关开发，笔者更认为是更适合是Windows自己的编译器，其他只是它附带的。。。。。

  • 对于Visual Studio Code

优点

  1. 体积较小(相对前两者)，不占空间。笔者安装上就觉得，体积不大，可以接受。启动速度不错，可以！
  2. 也有丰富的插件库，编程插件看需下载。有微软官方的插件库，也用第三方开发者开发的插件库。看自己需要啥，自己下载配置，自定义。

缺点

  1. 不像Pycharm里面基本配置都铺好了，一切从语言开始配置！不配置的话，基本的像一个带皮肤的的记事本。默认是英文界面，可以换成中文。
  2. 因为可以自定义，所以你要自定义！你得根据你的编程语言去插件商店里下载对应插件。否则，之后的运行很麻烦。这点新手不太友好，可对过来人讲，这也很适合新手！！！
  • 综其上所述，字里长短，你也知道笔者使用和推荐的是哪个吧！
